 ## Project Overview
 A console-based todo app written in python for managing tasks.
 User can:
 - See Tasks
 - Add Tasks
 - Remove Tasks


## Algorithm Summary

```text
1. Start
2. Choice Menu appears
3. Take input from user for choice
4. Show Output based on his choice:
- Search / Add / Remove Tasks / Exit 

```
